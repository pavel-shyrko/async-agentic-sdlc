Implement a geometric shapes utility package.
1. Create a base abstract class `Shape` in `src/geometry/base.py` with an abstract method `area()`.
2. Implement `Circle` and `Rectangle` subclasses in `src/geometry/shapes.py`.
3. Implement a volume calculator for standard 3D extensions (Cylinder, Cuboid) in `src/geometry/volume.py` that utilizes the 2D shapes logic.
All inputs must be strictly non-boolean numbers and greater than zero, handled by standard exceptions.
